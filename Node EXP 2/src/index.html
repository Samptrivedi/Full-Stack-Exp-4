Playing Card Collection API

This is a simple RESTful API for managing a collection of playing cards. Use an API client like Postman or Insomnia to test the endpoints.

GET /cards

Retrieves a list of all cards in the collection.

GET /cards/:id

Retrieves a single card by its unique ID. Example: /cards/1

POST /cards

Adds a new card to the collection. The request body must be JSON and contain suit and value properties.

Example Body: { "suit": "Clubs", "value": "Jack" }

DELETE /cards/:id

Deletes a card from the collection by its unique ID. Example: /cards/2
